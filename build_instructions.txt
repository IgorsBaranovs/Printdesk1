
ANDROID APK BUILD (Capacitor)

1) Install Node.js
2) npm install -g @capacitor/cli

Inside project:

npm init -y
npm install @capacitor/core @capacitor/android

npx cap init PrintDesk com.printdesk.app
npx cap add android

Copy files to:
www/

Then:
npx cap sync
npx cap open android

Open in Android Studio:
- Build → Build APK
